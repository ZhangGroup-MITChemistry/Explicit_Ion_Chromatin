dihedral_style 3spn2 command

Syntax: 

dihedral_style 3spn2

Examples:

  • dihedral_style 3spn2
  • dihedral_coeff 1 1.434 -359.17 0.30

Description:

The 3spn2 dihedral style uses the potential

and was originally implemented for inclusion in the 3SPN.2 coarse-grained DNA model of (Hinckley) .

The following coefficients must be defined for each dihedral type via the dihedral_coeff command as in the example above, or in the data file or restart files read by the read_data or read_restart commands.

  • K (energy)
  • phi0 (degrees)
  • sigma (degrees)

Restrictions:

This dihedral style can only be used if LAMMPS was built with the USER-3SPN2 package. See the Making LAMMPS section for more info on packages.

Related commands:

dihedral_coeff

Default: none

(Hinckley) Hinckley, Freeman, Whitmer and de Pablo. J Chem Phys, 139, 144903 (2013).
